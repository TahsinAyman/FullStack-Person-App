    <h1>Prerequisite</h1>
    <ul>
      <li>NodeJS</li>
      <li>Java</li>
      <li>MySQL</li>
    </ul>
    <h1>Frontend</h1>
    <p>To Run the Frontend: </p>
    <ul>
      <li>cd frontend</li>
      <li>npx serve -s build</li>
    </ul>
    <p>Frontend Server: </p>
    <ul>
      <li>host: localhost</li>
      <li>port: 8080</li>
    </ul>
    <h1>Backend</h1>
    <p>To Run the Backend: </p>
    <ul>
      <li>cd backend</li>
      <li>mvn clean install</li>
      <li>java -jar target/backend-1.0.0.jar</li>
    </ul>
    <p>Backend Server: </p>
    <ul>
      <li>host: localhost</li>
      <li>port: 8000</li>
    </ul>
